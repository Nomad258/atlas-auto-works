## Option 2: Use Sketchfab Search (Most Reliable)

Since direct links may change, use this method:

1. **Go to Sketchfab**: https://sketchfab.com
2. **Search**: "{car name} downloadable" (e.g., "Porsche 911 downloadable")
3. **Filter**: Check "Downloadable" on the left sidebar
4. **Download**: Click model → Download → Select "glTF (.glb)"
5. **Save as**: `{make}-{model}.glb` (lowercase, hyphens)

### Search Terms to Use:
- "Porsche 911 downloadable glb"
- "BMW M3 downloadable glb"
- "Mercedes C63 downloadable glb"

## Troubleshooting

**Model too big?**
- Compress using: https://gltf.report/
- Or use Blender: File → Export → glTF 2.0 → Check "Apply Modifiers"

**Wrong scale?**
- Edit `Car3DViewer.jsx` line ~30: change `scale={2}` to adjust

**Can't find model?**
- Check console for errors
- Verify filename matches exactly
- File must be in `/public/models/` not `/models/`
